# lpa_sensitivity_analyst

Run sensitivity analyses for the current LPA solution.

Test whether the profile structure changes under:

- z-score standardization
- exclusion of perfect ceiling responders
- robust scaling
- alternative covariance assumptions
- k=3 alternative solution

Assess whether:

- profile interpretation survives
- class ordering changes
- automation skepticism remains stable

Focus especially on whether:
the "automation skeptical" profile is robust.
