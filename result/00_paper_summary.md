# 논문 전략 요약

## 1. 현재 최종 전략

- 조직 맥락 변수는 `Q16_1~Q16_4` 평균으로 고정
- `support_main`은 organizational AI support climate의 observed index로 해석
- Q20은 composite으로 묶지 않음
- `Q20_1~Q20_3`를 main item-level outcome으로 분석
- `Q20_4`는 supplementary outcome으로 분리

## 2. 분석식

모든 결과항목에 대해 동일한 식을 적용한다.

- `effect ~ motivation + support_main + gender + rank_code + career_code`
- `Q20_k ~ motivation + support_main + gender + rank_code + career_code`
- `Q20_k ~ motivation + support_main + effect + gender + rank_code + career_code`

여기서 `k = 1, 2, 3`이고, `Q20_4`는 supplementary로만 별도 제시한다.

## 3. 결과 요약

- `Q20_1`: motivation 중심
- `Q20_2`: motivation + effect 중심
- `Q20_3`: support_main + effect 중심
- `Q20_4`: supplementary에서 support_main 중심

## 4. 논문 메시지

본 연구는 생성형 AI 기대를 단일한 척도로 가정하지 않고, 서로 다른 기대 유형 및 인식을 개별 결과항목으로 분석한다. 분석 결과, 자발적 AI 활용동기와 조직 AI 지원 분위기는 모든 기대에 동일하게 작동하지 않았으며, 기대의 내용영역에 따라 서로 다른 경로를 통해 영향을 미치는 것으로 나타났다.
