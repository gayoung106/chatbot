# 03 Descriptive Analysis for AI Users

## Sample overview

- Full sample N = 1608
- AI users N = 377
- Non-users N = 1231

## Scale construction used in the main study

- motivation = mean(Q9_3, Q9_4)
- effect = mean(Q7_1~Q7_5)
- support_main = mean(Q16_1~Q16_4)
- main strategic expectancy outcomes = Q20_1, Q20_2, Q20_3
- supplementary outcome = Q20_4
- Q16_5~Q16_7 are excluded from the main organizational-support index because they blend organizational and personal-level evaluation content.

## Main-study interpretation

- motivation is represented with a two-item observed index, so average inter-item correlation is more informative than coefficient alpha.
- effect and support_main show stable descriptive distributions among AI users.
- Q20_1~Q20_3 can be compared item-by-item without forcing them into a single composite.
- Q20_4 is retained only as a supplementary threat-perception outcome, not as part of the main strategic expectancy block.
